# 🎮 Number Guessing Game

A fun console-based game built in C++ where the player
guesses a randomly generated number with AI hints.

## Features
- Random number generation
- AI hints (Odd/Even + Range)
- Scoring system based on attempts
- Play again option

## How to Run
1. Clone the repository
2. Open in any C++ compiler
3. Compile and run main.cpp

## Technologies Used
- C++
- Standard Libraries (cstdlib, ctime)

## What I Learned
- Random number generation in C++
- Loops and conditionals
- Game logic design
